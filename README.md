## Features

- **GFM**: GitHub Flavored Markdown + alert call-outs

- **Preview**
- **No telemetry**
- **Offline**
- **No ads**
- **Free**
- **Split view**
- **Hotkeys**
- **Import/Export**
- **Theme**
- **Spellcheck**
